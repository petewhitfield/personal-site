# Personal Website

Simple AI & software engineering blog, built with SvelteKit.

## Development

```sh
# Format the codebase with Prettier
bun run format

# Run Prettier checks and ESLint
bun run lint

# Run the test suite
bun run test

# Run SvelteKit sync and type/content checks
bun run check
```

## Deployment

This site is deployed with Kamal. Run the following command to deploy. You will need the local `.env` from a team member.

```sh
kamal deploy
```
