# personal-site

My personal website, built with SvelteKit.

## Development

```sh
# Format the codebase with Prettier
bun run format

# Run Prettier checks and ESLint
bun run lint

# Run the test suite
bun run test

# Run SvelteKit sync and type/content checks
bun run check
```

## Deployment

This site is deployed with Kamal.

```sh
kamal deploy
```
